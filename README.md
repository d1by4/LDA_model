# Topic Modeling LDA - Bahasa Indonesia

Proyek ini bertujuan untuk melakukan pemodelan topik (Topic Modeling) menggunakan algoritma **Latent Dirichlet Allocation (LDA)** pada dokumen teks berbahasa Indonesia.

## 🚀 Fitur Utama

- **Preprocessing**: Pembersihan teks, tokenisasi, dan stemming menggunakan `Sastrawi`.
- **LDA Model**: Implementasi model LDA menggunakan `Gensim`.
- **Visualisasi**: Visualisasi interaktif menggunakan `pyLDAvis` dan `WordCloud`.
- **Notebook Support**: Dapat dijalankan di Jupyter Notebook atau VS Code.

## 🛠️ Prasyarat

Pastikan Anda sudah menginstal Python (disarankan versi 3.9 - 3.11).

### Setup Environment

1. Buat virtual environment:
   ```bash
   python -m venv lda_env
   ```
2. Aktifkan environment:
   ```bash
   Windows: .\lda_env\Scripts\activate
   Mac/Linux: source lda_env/bin/activate
   ```
3. Instal library yang diperlukan:
   ```bash
   pip install -r requirements.txt
   ```

### Tips Tambahan:

Karena Anda sudah memiliki daftar library yang panjang di pesan sebelumnya, sebaiknya Anda simpan daftar tersebut ke dalam file bernama `requirements.txt`.

**Caranya:**

1. Buat file baru bernama `requirements.txt`.
2. Tempel (paste) daftar library yang Anda berikan tadi ke dalam file tersebut.
3. Simpan di folder yang sama dengan `README.md`.
